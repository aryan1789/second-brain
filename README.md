# Second Brain Vault

A personal knowledge management system for capturing, organizing, and surfacing insights across multiple domains.

## Structure

- **raw/** — Source documents and exports (read-only, never modified by system)
- **wiki/** — Processed knowledge base maintained by Claude (concepts, projects, people)
- **journal/** — Daily notes and reflections
- **content/** — Content pipeline and outputs

## Quick Reference

- Start here: [wiki/index.md](wiki/index.md) — master catalog
- Activity log: [wiki/log.md](wiki/log.md) — append-only record of changes
- Daily work: [journal/](journal/) — date-based entries

## Setup

Each directory contains a README explaining what belongs there and where to export source material from.
