# COMP838 - Introduction to Deep Learning

AUT course (completed). These notes capture the durable machine learning / deep learning concepts taught — architectures, definitions, and techniques — for ongoing reference. Course administration, assessment structure, detailed historical timelines, and lecturer-specific research projects have been intentionally left out as no longer relevant now the course is finished.

## Topic Areas Covered So Far

- Deep learning fundamentals: stacked nonlinear modules, selectivity vs. invariance
- Core architectures: ANNs, CNNs, RNNs, DNNs, MLPs
- ML vs DL distinction
- Data augmentation techniques
- Standard neural network training workflow (train/val/test split, epoch/batch/iteration)
- Clustering (SOM) and time-series (NARX) architectures
- Training mechanics: backpropagation, SGD, activation functions, loss functions, softmax
- Model evaluation: overfitting/underfitting, confusion matrix metrics, IoU
- Theory: Universal Approximation Theorem
- CNN spatial mechanics: receptive field, stride, padding
- Object detection: one-stage vs two-stage, anchor boxes, R-CNN family, YOLO (v1-v4), SSD
- Very deep CNN architectures: DenseNet, ResNet (residual learning, degradation problem)
- Pretrained ImageNet classifiers: AlexNet, VGG-19, GoogLeNet, Inception-v3
- Time series regression and forecasting with LSTM
- Sequence modelling: RNN recap, LSTM gating mechanism
- Transformers and the Vision Transformer (ViT)
- Generative Adversarial Networks (generator/discriminator, minimax training)
- Reinforcement learning (agent-environment loop, policy, RL workflow)
- Transfer learning (categories, approaches, pretrained-network fine-tuning)
- Ensemble learning (diverse base learners, combination strategies)
- Distributed/parallel training (data parallelism, batch size/learning rate scaling, cloud computing, Google Colab)
