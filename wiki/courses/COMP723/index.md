# COMP723 - Data Mining and Machine Learning

AUT course (notes complete — Lectures 1-6 and 8-11, no Lecture 7), lecturer Parma Nand. First 6 weeks cover structured data mining (mostly numeric data); the second 5 weeks cover unstructured data mining (text/speech). These notes capture the durable data mining and machine learning concepts taught — algorithms, evaluation methods, and techniques — for ongoing reference. Course administration (staff contacts, lecture/lab timetables, assessment weightings) has been intentionally left out as not durably useful.

## Topic Areas Covered

- Data Mining fundamentals: data vs. knowledge, the Knowledge Discovery (KDD) process
- Classification basics: training/test sets, structural descriptions (rules, decision trees), numeric prediction (linear regression)
- Bias in machine learning: algorithm (inductive) bias vs. data bias
- Data preprocessing and data quality: noise, missing values, duplicates, outliers, normalization, balancing, feature selection
- Evaluation: confusion matrix, precision/recall/F1, sensitivity/specificity, Fβ, association rules, clustering
- Classification algorithms: Naive Bayes, Decision Trees (information gain, entropy, pruning)
- Overfitting and underfitting in decision trees
- K-Nearest Neighbors (instance-based learning)
- Bias-variance trade-off and ensemble learning (bagging, boosting, AdaBoost)
- Cross-validation and ROC/AUC evaluation
- Artificial Neural Networks: McCulloch-Pitts neurons, perceptrons, MLP training and parameters
- Unsupervised clustering: K-means, distance measures, the Elbow Method
- Sound/speech processing: sampling, librosa, spectrograms/MFCC, ASR
- NLP preprocessing: tokenization (token/type/term), stop words, normalization, lemmatization/stemming, Porter's Algorithm
- Text classification and vectorisation: bag-of-words, feature selection/construction, Mutual Information, TF-IDF
- Word embeddings: CBOW, Skip-gram, Word2Vec, GloVe
- Transformers: RNN language models, encoder-decoder, attention, self-attention (QKV), multi-head attention, positional encoding
- Information extraction: coreference, taxonomic relations, Hearst patterns, supervised and bootstrapped relation extraction
