## Overview

A student skill-exchange platform connecting peers for collaborative learning and capability development. Built as a full-stack web application to explore diverse tech stacks and frameworks.

## Purpose

- Enable students to exchange skills and knowledge
- Create a networking platform for peer learning
- Explore innovative networking concepts
- Practice full-stack development across multiple tech stacks

## Technical Stack

- **Frontend:** React with TypeScript
- **Backend:** ASP.NET
- **Database:** Supabase (PostgreSQL)
- **Authentication:** JWT with Supabase
- **Styling:** Custom UI with design refinements

## Core Features

- User profiles with skill listings
- Peer review and rating system
- Project showcase and collaboration tools
- Label-based filtering for skill discovery
- Real-time UI updates for user interactions

## Development Challenges

- JWT authentication setup and configuration
- Database connection troubleshooting
- Component import resolution in TypeScript
- SearchBar UI/UX refinement
- Multi-label filtering implementation
- User list sorting and data management
