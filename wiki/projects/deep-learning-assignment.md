## Overview

A final-year software engineering assignment focused on deep learning. The project involves comprehensive exploration of deep learning techniques, model selection, and practical application to real-world classification problems.

## Purpose

- Build depth in machine learning theory and application
- Explore different architectures and approaches
- Practice critical analysis and experimental design
- Apply concepts to novel datasets

## Key Topics Covered

- Deep learning frameworks and libraries
- Transfer learning techniques
- YOLO object detection
- Dataset selection and preparation
- Model optimization and tuning
- Evaluation metrics and validation

## Notable Applications

### Kiwifruit Ripeness Classification
- Using YOLOv4 and transfer learning
- RGB data-based classification
- Real-world agricultural computer vision application

### EMG Sensor Data Analysis
- Deep learning on biometric signals
- Multi-agent systems for data processing
- OpenBCI sensor integration

## Learning Approach

- Leverage lecture notes and tutorial problems
- Practice problem-solving iteratively
- Refine answers with critical feedback
- Build understanding across multiple domains
