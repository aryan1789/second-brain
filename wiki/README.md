# Wiki

The processed knowledge base — your curated, interconnected notes system. **This is Claude's domain to create and maintain.**

## Structure

- **index.md** — Master catalog of all wiki pages and topics
- **log.md** — Append-only activity log of changes and updates
- **concepts/** — Concept and topic pages; reusable knowledge
- **projects/** — Project-specific pages; goals, status, artifacts
- **people/** — People dossiers; contacts, collaborators, references

## Principles

- Pages are interconnected via wikilinks `[[page-name]]`
- Concepts are timeless; projects are time-bound
- All content is processed and synthesized from raw/ sources
- log.md tracks what's new and changed

## Workflow

1. New insights from raw/ are processed into concept or project pages
2. Relationships and links are maintained
3. index.md stays current as the source of truth
4. log.md records all significant changes

## Getting Started

- Start here: [index.md](index.md)
- Recent changes: [log.md](log.md)
