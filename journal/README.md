# Journal

Daily notes and reflections. Personal logs and journaling space.

## Purpose

- Daily reflections and check-ins
- Thoughts and observations
- Personal progress tracking
- Emotional processing and reflections

## Organization

Organize by date: `DD-MM-YYYY.md`

Examples:
- `2026-06-12.md`
- `2026-06-11.md`

## File Structure

Optional template:
```markdown
# 2026-06-12

## Thoughts
- [your reflection here]

## Today's Wins
- [accomplishment]

## Tomorrow's Focus
- [intention]

## Notes
- [anything else]
```

## Workflow

- One file per day
- Review periodically for patterns and insights
- Some reflections may inspire new concept or project pages
